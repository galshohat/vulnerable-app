# InboxOps

> ⚠️ **WARNING**: This application is **deliberately insecure**. It is a training
> target for a security workshop. **Do NOT deploy it anywhere.**

InboxOps is a fake email-marketing SaaS platform built with Python/Flask.
It includes authentication, payments, file handling, API endpoints, database
access, and deployment configs (Docker, Terraform, GitHub Actions).

This repository is used as a read-only scan target — AI agents analyze the
codebase and produce a security audit report.
